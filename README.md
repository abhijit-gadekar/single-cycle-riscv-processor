# Single-Cycle RISC-V Processor

A 32-bit Single-Cycle RISC-V Processor implemented in Verilog HDL following the architecture presented in:

> **Digital Design and Computer Architecture: RISC-V Edition**  
> Sarah L. Harris and David Money Harris

---

## Project Overview

This project implements a complete RV32I Single-Cycle Processor from scratch using Verilog HDL.

The processor executes every instruction in a single clock cycle and serves as a foundation for understanding:

- RTL Design
- Computer Architecture
- Processor Design
- Digital IC Design
- ASIC Design Flow
